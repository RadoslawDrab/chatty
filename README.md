## How to use

#### Creating a new project

- Install [Expo App](https://expo.dev/) on mobile device
- Clone repository: `git clone https://github.com/RadoslawDrab/chatty.git`
- `cd` into the project
- Run `npm install` or `yarn install`

#### Starting

```sh
npm start
```

or

```sh
yarn start
```

## Dependencies

- [TypeScript](https://www.typescriptlang.org/)
- [React Native](https://reactnative.dev/)
